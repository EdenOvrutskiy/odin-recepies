# odin-recepies
Goal: Create a webpage using html + css.

Theme: recipes

Html components: 
* the boilerplate (!doctype, html , head, body 
tags, encoding, lan)
* body elements / tags: 
  * headers + size: h1~h6
  * paragraphs / separate bodies of text: p
  * bold / italics: strong, em
  * links: a + href attribute
  * images: img + src attribute
  * lists:
    * ordered: ol
    * unordered: ul
    * items in lists: li
